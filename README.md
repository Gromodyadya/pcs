  Было создано 3 экрана: Welcome, Main Screen, Add Item Screen. 
    
  Использовались следующие элементы интерфейса: кнопки, текстовые поля, иконки.
    
  Навигация оранизованна следующим образом: Кнопка "Начать" на странице приветсвия перенапрявляет пользователя на основную страницу, кнопка "+" на основной странице перенаправялет на страницу добавления заметки, кнопки "Сохранить" и "Назад" на страницу добавления заметки перенаправляют на основную страницу.
    
  Данный интерфейс удобен простотой и ясностью для пользователя, но можно поработать над цветами и расположением элементов, для более приятной картинкти для глаз пользователя. Так же нужно добавить кнопку удаления заметок.
