Для создания интерфейса приложения были использованы следующие виджеты:

MaterialApp: Корневой виджет, который предоставляет множество стандартных функций для приложения, таких как навигация и темы.
Scaffold: Является основным каркасом для экрана. Он предоставляет готовые слоты для AppBar, body и других стандартных элементов интерфейса.
AppBar: Верхняя панель приложения, в которой был размещен заголовок "Практика 3".
Column: Виджет компоновки, который располагает дочерние элементы в вертикальной последовательности. Он был использован как основной контейнер в body для размещения текста, контейнера, кнопки и ряда с иконками.
Text: Отображает строку текста. Использовался для вывода приветствия.
Container: Универсальный прямоугольный виджет, которому можно задать цвет, размеры, отступы и другие декорации. В работе использовался для создания цветного блока.
ElevatedButton: Стандартная кнопка с тенью и заливкой.
Row: Виджет компоновки, который располагает дочерние элементы в горизонтальной последовательности. Был использован для размещения двух иконок рядом.
Icon: Виджет для отображения иконок из стандартной библиотеки Material Design.
SizedBox: Невидимый виджет, который используется для создания пустого пространства (отступов) между другими виджетами.

Цвета задавались двумя основными способами:

Использование стандартной палитры Colors: Для большинства элементов, таких как фон AppBar (Colors.blue) или Container (Colors.green), использовались предопределенные константы из класса Colors.
Использование HEX-кода: Для одной из иконок был применен кастомный цвет через конструктор Color(0xFF...).

Для стилизации текста использовался параметр style виджета Text, который принимает объект TextStyle. Внутри TextStyle были настроены следующие параметры:

fontSize: для установки размера шрифта.
fontWeight: для изменения жирности текста.
color: для задания цвета текста.

В процессе выполнения работы возникли некоторые трудности:

Поначалу было сложно привыкнуть к тому, что каждый элемент интерфейса является виджетом, и они вкладываются друг в друга, образуя дерево.
Возникли сложности с центрированием всех элементов по вертикали и горизонтали. Это было решено с помощью свойства mainAxisAlignment: MainAxisAlignment.center у виджета Column и оборачивания всего Column в виджет Center.
