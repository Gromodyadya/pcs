Для создания интерфейса приложения были использованы следующие виджеты:

StatefulWidget: Основной виджет для экрана, так как интерфейс должен был динамически изменяться (обновлять текст счётчика) в ответ на действия пользователя.
Scaffold: Предоставил базовую структуру экрана, включая AppBar и body.
Column: Использовался для вертикального расположения элементов: текста и двух кнопок.
Center: Применялся для центрирования Column на экране, что позволило выровнять все элементы по центру.
Text: Отображал текущее значение счётчика.
ElevatedButton: Стандартная кнопка, использовалась для реализации функции "Сбросить".
GestureDetector: Специальный виджет, который отслеживает различные жесты. Он был использован для кнопки "Увеличить", чтобы реализовать два разных действия: на обычное нажатие (onTap) и на долгое нажатие (onLongPress).
Container: Применялся для стилизации кнопки "Увеличить", чтобы задать ей фон, цвет и внутренние отступы.
SizedBox: Использовался для создания фиксированного пустого пространства (вертикальных отступов) между элементами в Column.

Управление состоянием было реализовано следующим образом:

Создание переменной состояния: В классе _State, который относится к StatefulWidget, была объявлена приватная переменная int _counter = 0;. Она хранит текущее значение счётчика.
Использование setState(): Любое изменение переменной _counter происходило внутри вызова функции setState(). Этот метод сообщает Flutter, что внутреннее состояние объекта изменилось, и виджет необходимо перерисовать, чтобы отобразить актуальные данные.
Обновление интерфейса: При вызове setState(), Flutter перестраивает дерево виджетов. В моем случае, он заново вызывает метод build, и виджет Text('Значение счётчика: $_counter') отрисовывается уже с новым значением переменной _counter.

В приложении была реализована обработка следующих пользовательских событий:

Для кнопки "Увеличить" использовался onTap виджета GestureDetector. При его срабатывании счётчик увеличивался на 1.
Для кнопки "Сбросить" использовался стандартный onPressed виджета ElevatedButton. Он обнулял значение счётчика.
Долгое нажатие (onLongPress):
Для кнопки "Увеличить" также был реализован onLongPress виджета GestureDetector. При долгом удержании пальца на кнопке счётчик увеличивался сразу на 10.
